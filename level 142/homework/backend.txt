რა არის backend?
backend არის საიტი უკანა და დამალული ნაწილი, სადაც ეშვება სერვერი და frontendში მომხდარი რაღაც მოვლენის შემდეგ გაიგზავნება მოთხოვნდა backendში და მოხდება გარკვეული მოვლენა
it's types - მაგალითად express.js და node.js

request & response - რექუესთი არი მოთხოვნა, რომელიც იგაზავნება frontendიდან backendში კონკრეტულად სერვერში
response არის რექუესთის პასუხი, რომელიც სერვერიდან მოდის frontendში

database - არის მონაცემთა ბაზა, რომელშიც ინახება მომხმარებლის მონაცემები
მისი ტიპები - sql ანუ reltional - არის ცხრილის სახით მონაცემების ერთობლიობა
nosql - ეს გვიჩვენებს როგორც jsonის სახით

connection among frontend, server and database 
frontendიდან გაგზავნილი რექუესთი მიდის სერვერში, შემდეგ სერვერიდან უკავშირდება databaseს
