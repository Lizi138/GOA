NoSQL ბაზები არის მონაცემთა ბაზები, რომლებიც არ იყენებენ SQL ცხრილებს

Document-oriented database
მონაცემები ინახება დოკუმენტება, თითო დოკუმენტი შეიძლება განსხვავებული სტრუქტურის იყოს
მარტივია რეალური ობიექტების შენახვისთვის

Key-Value database
ყველაფერი ინახება როგორც key-value წყვილი.
ძალიან სწრაფია და
მარტივი სტრუქტურა აქვ

Graph-based database
მონაცემები ინახება როგორც nodes

სწრაფი და რთული კავშირების მოძებნისთვის არის კარგი

Column-oriented database
მონაცემები ინახება columns-ებად და არა rows-ებად (როგორც SQL-ში)

ძალიან სწრაფია დიდი მოცულობის მონაცემებზე
